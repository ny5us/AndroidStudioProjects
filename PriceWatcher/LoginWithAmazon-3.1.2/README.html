Login with Amazon SDK for Android Readme

The Login with Amazon SDK for Android provides a library and documentation for developers to build connected mobile applications using Login with Amazon.

The Login with Amazon SDK for Android supports apps using Android 4.1 and higher (API Level 16)

After upgrading to version 3.1.2, if you later decide to roll back to the previous version of LWA, any users will need to either clean their app data or uninstall and reinstall the app.

For more information about using the SDK, please see Getting Started with Login with Amazon for Android Apps
